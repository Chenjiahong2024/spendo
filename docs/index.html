Spendo · 极简智能记账 App

用最简单的方式，掌控每一笔收支

Spendo 专为 iOS 打造，支持语音记账、小票扫描、智能分类、多账户和预算管理， 让记账变得轻松、有趣又安全。

快速记账

三种高效方式，随时随地记录收支。

  • 手动输入：几步完成一笔收支记录
  • 语音记账：说出「午饭 50 块」即可自动识别
  • 扫描小票：拍照识别金额，省去手工输入

智能分析与预算

用清晰的图表和预算工具，看懂你的钱花到哪里去了。

  • 收支统计饼图与趋势折线图
  • 支持按周、月、年多维度分析
  • 总预算与分类预算、超支提醒

多账户与多币种

管理现金、银行卡以及电子钱包，资产一目了然。

  • 支持多账户记账与账户间转账记录
  • 多币种支持，适合境外出行和多币种消费
  • 自动计算账户余额，资产结构更清晰

本地存储，重视隐私

Spendo 不依赖云端服务器，所有数据仅保存在你的设备本地。

  • 记账数据保存在本机，不上传到远程服务器
  • 使用系统级加密能力和面容 ID 保护隐私
  • 无需注册账号，即装即用
